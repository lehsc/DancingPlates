# DancingPlates
This is a simplified version of the game Dancing Plates, released for the Atari 2600 in 1982. The objective is to balance plates on sticks, similar to circus performers, while preventing them from falling. Here, the plates are represented by rectangles and the character is a triangle. You can only move the triangle sideways, press space to balance a rectangle (after positioning yourself just below the bar that supports it), and P to pause the game. Have fun!
